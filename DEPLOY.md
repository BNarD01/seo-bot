# SEO Bot - Deploy to Vercel

## 步骤

### 1. 推送到 GitHub
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/yourusername/seo-bot.git
git push -u origin main
```

### 2. 部署到 Vercel
1. 访问 https://vercel.com
2. 用 GitHub 登录
3. 导入项目
4. 设置环境变量：
   - `OPENAI_API_KEY` = your-key
5. 点击 Deploy

### 3. 绑定域名（可选）
1. 在 Vercel 添加自定义域名
2. 或在 Namecheap 买域名指向 Vercel

## 完成！

网站上线后就可以开始推广获客了！
