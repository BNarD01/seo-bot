# SEO Bot

AI-powered SEO article generator.

## Setup

```bash
# Install dependencies
pip install flask openai

# Set OpenAI API key
export OPENAI_API_KEY=your-key-here

# Run
python app.py
```

## Features

- Generate SEO-optimized articles
- Keyword research
- Multiple article lengths
- Ready-to-deploy
